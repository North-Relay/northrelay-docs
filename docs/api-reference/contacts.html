Contacts API

The Contacts API lets you store recipient information, organize contacts into lists, and manage tags for targeted email campaigns.

List Contacts

Retrieve all contacts for your account.

Query Parameters

ParameterTypeDefaultDescription
pageinteger1Page number
limitinteger25Items per page (max 100)
searchstring--Search by email or name
listIdstring--Filter by contact list ID
tagstring--Filter by tag key (e.g., tag=region)

Bulk Import Contacts

Import multiple contacts in a single request. Existing contacts (matched by email) are updated; new contacts are created.

Request Body

FieldTypeRequiredDescription
contactsobject[]YesArray of contact objects (max 10,000)
listIdstringNoAdd all imported contacts to this list
updateExistingbooleanNoUpdate existing contacts on email match (default: true)

Each object in the contacts array uses the same fields as Create Contact.

Example Response

If individual contacts fail validation, they are skipped and listed in the errors array with the reason.
