Sending API

Send single emails, batch emails, scheduled emails, and validate email addresses via the NorthRelay API.

Sending API

The Sending API provides four endpoints for email delivery and address validation. All sending is API-only -- direct SMTP AUTH access is not available.

Send Email

Send a single email to one or more recipients.

Request Body

FieldTypeRequiredDescription
fromEmailAddressYesSender address
toEmailAddress[]YesRecipients (1--1,000)
ccEmailAddress[]NoCC recipients
bccEmailAddress[]NoBCC recipients
replyToEmailAddressNoReply-to address
contentContentYesEmail content (subject, body, or template)
variablesRecord<string, string>NoTemplate variables for {{variable}} substitution
themeIdstringNoTheme ID for styled rendering
categoriesstring[]NoUp to 10 categories (alphanumeric, hyphens, underscores; max 64 chars each)
tagsRecord<string, string>NoCustom key-value tags for filtering in analytics
metadataRecord<string, any>NoArbitrary metadata returned in webhooks
attachmentsAttachment[]NoFile attachments

Batch Send

Send personalized emails to up to 1,000 recipients in a single request. Each recipient can have unique variable substitutions.

contentContentYesShared email content; subject is required (not optional)

tagsRecord<string, string>NoCustom key-value tags
metadataRecord<string, any>NoArbitrary metadata

Schedule Email

Schedule an email for future delivery. The request body is identical to Send Email with one additional required field.

Additional Field

FieldTypeRequiredDescription
scheduledForstring (ISO 8601)YesDelivery time; must be in the future and within 1 year

Cancelling a Scheduled Email

Cancel a scheduled email before it is sent. Returns 404 if the email has already been delivered.

Validate Email

Check whether an email address is deliverable without sending a message. Performs syntax validation, DNS MX lookup, and optionally SMTP handshake verification.

Request Body

FieldTypeRequiredDefaultDescription
emailstringYes--Email address to validate
extendedbooleanNofalseEnable SMTP handshake check (slower but more accurate)
timeoutintegerNo10000Timeout in milliseconds (1,000--60,000)

Example Request

Example Response

When validation fails, the suggestion field may offer a corrected address (e.g., "suggestion": "alice@gmail.com" for a misspelled domain).

Bulk Validate

Validate up to 10,000 addresses in a single request.

FieldTypeRequiredDefaultDescription
emailsstring[]Yes--Email addresses to validate (1--10,000)
extendedbooleanNofalseEnable SMTP handshake checks
timeoutintegerNo30000Timeout in milliseconds (1,000--60,000)
continueOnErrorbooleanNotrueContinue validating remaining addresses if one fails
