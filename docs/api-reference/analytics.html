Analytics API

The Analytics API provides access to delivery metrics, engagement data, geographic breakdowns, and provider-level statistics. Use these endpoints to monitor your sending performance and optimize your email strategy.

Heatmap

The rates object contains percentages. clickToOpenRate is clicks divided by opens (not by sends).

Metric Definitions

MetricDefinition
sentTotal emails submitted to the delivery queue
deliveredEmails accepted by the recipient mail server (2xx SMTP response)
bouncedEmails permanently rejected (5xx SMTP response)
deferredEmails temporarily rejected; NorthRelay will retry automatically
openedTotal open events (includes repeat opens by the same recipient)
uniqueOpensDistinct recipients who opened at least once
clickedTotal click events (includes repeat clicks)
uniqueClicksDistinct recipients who clicked at least once
complainedRecipients who marked the email as spam
unsubscribedRecipients who clicked the unsubscribe link
deliveryRatedelivered / sent * 100
bounceRatebounced / sent * 100
openRateuniqueOpens / delivered * 100
clickRateuniqueClicks / delivered * 100
clickToOpenRateuniqueClicks / uniqueOpens * 100
complaintRatecomplained / delivered * 100
unsubscribeRateunsubscribed / delivered * 100
