MCP Server

The NorthRelay MCP Server (@northrelay/mcp-server v1.1.1) exposes the full NorthRelay API as tools that AI agents can use directly. It implements the Model Context Protocol (MCP) standard, allowing AI assistants to send emails, manage templates, check delivery status, and more through natural language.

What is MCP?

The Model Context Protocol (MCP) is an open standard that lets AI applications interact with external services through structured tool calls. Instead of writing code, you describe what you want in plain language, and the AI agent calls the appropriate NorthRelay tools on your behalf.

Supported AI clients:

  • Claude Desktop
  • Cursor
  • Windsurf
  • Any MCP-compatible client

NORTHRELAY_BASE_URLNohttps://app.northrelay.caCustom API base URL

Claude Desktop

Add the following to your Claude Desktop configuration file:

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json Windows: %APPDATA%\Claude\claude_desktop_config.json Linux: ~/.config/Claude/claude_desktop_config.json

Restart Claude Desktop after saving the configuration.

Cursor

In Cursor settings, add an MCP server with:

  • Name: northrelay
  • Command: npx @northrelay/mcp-server
  • Environment: NORTHRELAY_API_KEY=nr_live_your_api_key_here

Usage Examples

Once configured, interact with NorthRelay through natural language in your AI client.

Sending Emails

"Send a welcome email to jane@example.com with the subject 'Welcome to our platform' and an HTML body with a greeting."

The AI agent will call send_email with the appropriate parameters.

Managing Templates

"Create a new email template called 'Password Reset' with a subject line 'Reset your password' and a body that includes a reset link placeholder."

"List all my email templates."

"Delete the template called 'Old Newsletter'."

Checking Delivery Status

"What's the delivery status of the last 10 emails I sent?"

"Show me the delivery analytics for March 2026."

Domain Management

"Add the domain mail.mycompany.com and show me the DNS records I need to configure."

"Verify my domain mail.mycompany.com."

Reputation Monitoring

"What's my current sender reputation score?"

"Show me any active reputation incidents."

"Get my reputation history for the past 30 days."

Campaign Management

"Create a campaign called 'Spring Sale' using my 'promotion' template, targeting my 'active-customers' contact list, scheduled for March 25th at 10am UTC."

"What are the stats for my 'Spring Sale' campaign?"

Troubleshooting

Server Not Starting

Test the server manually:

Tools Not Appearing in AI Client

  1. Restart your AI client after updating the configuration
  2. Verify the configuration file path is correct for your OS
  3. Check that npx is available in your system PATH

Authentication Errors

Ensure your API key:

  • Starts with nr_live_ (production) or nr_test_ (testing)
  • Has not been revoked
  • Has the necessary permissions for the operations you are performing
