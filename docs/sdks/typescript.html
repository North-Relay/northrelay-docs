TypeScript SDK

The official NorthRelay TypeScript SDK (@northrelay/sdk v1.5.1) provides full access to the NorthRelay API with complete type safety, automatic retries, and built-in webhook verification.

Automatic Retries

The SDK automatically retries failed requests with exponential backoff:

  • Retries on 5xx server errors and network timeouts
  • Does not retry on 4xx client errors (validation, auth, not found)
  • Backs off exponentially: 1s, 2s, 4s (with maxRetries: 3 and retryDelay: 1000)
