Alert Recommendations

Set up alerts for these critical conditions:

Tier 1: Page Immediately

  • Health check endpoint returns non-200 for > 2 minutes
  • Mail queue size > 500 and growing
  • Disk usage > 90%
  • PostgreSQL connection count > 95
  • WireGuard handshake > 5 minutes old

Tier 2: Alert Within 15 Minutes

  • Bounce rate > 10% over 1 hour
  • Delivery time p95 > 60 seconds
  • App error rate > 5% of requests
  • Redis memory > 90% of maxmemory

Tier 3: Review Daily

  • Bounce rate > 5% daily average
  • Queue size consistently > 50
  • Slow queries > 500ms appearing in PostgreSQL logs
  • Disk usage > 80%
